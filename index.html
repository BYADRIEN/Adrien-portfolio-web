A propos de moi

Bonjour, je suis Adrien Selle, étudiant en BES Web Developer à Namur, passionné par le développement web et les technologies innovantes. Formation et Expérience : Engagé dans le programme BES Web Developer, j'ai eu l'opportunité de consolider mes compétences en programmation, avec une expérience pratique variée. Mon parcours académique inclut un stage où j'ai travaillé sur un CMS, utilisant HTML, CSS, et JavaScript pour créer des solutions web efficaces et esthétiques. Expérience Professionnelle : Mon deuxième stage a été une immersion approfondie dans le développement, avec une concentration sur Angular et Java. Cette expérience m'a permis de plonger dans des technologies avancées, d'approfondir mes connaissances et de développer des compétences essentielles pour des projets de développement web.
